# 社区集市 
基于开源本地⽣活项⽬（黑马商城）进⾏⼆次开发，打造类 “⼤众点评” 的本地⽣活服务平台，覆盖⽤户⾜迹追踪、商家优惠券
创建与分发、优惠券秒杀抢券、下单等完整业务链路。

### 技术架构
SpringCloud Nacos OpenFeign MyBatis-Plus  MySQL Redis  Elasticsearch RocketMQ 

### 主要工作：
- 微服务拆分和引入
- 新的微服务设计：用户足迹追踪、商品ES搜索、优惠券秒杀哦
- 
### 技术亮点：
- 微服务设计：将原本的服务抽离成：用户管理、商品服务、购物车服务、订单服务、支付服务
- spring gateway：将用户身份信息的授权提前到gateway，使用JWT传递用户身份，解析用户身份到请求头，各个微服务使用hm-common中的拦截器直接获取用户信息。
- 用户足迹管理： 将用户对商品的浏览等行为抽离成单独的微服务，记录用户的足迹，并先存入Redis再定时扫描入库。
- 对商品的ES搜索：将商品搜索功能独立成单独的微服务，使用ElasticSearch进行商品的搜索，提升搜索性能和用户体验。
